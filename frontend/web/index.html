01 / About

A quiet control room for project work.

Core is a full-stack project management website built for group projects. Members can create shared projects, organize work into board columns, assign tasks, and keep comments attached to each card.

03 / Tech used

The stack behind Core.

Frontend

HTML, CSS, and vanilla JavaScript drive the responsive interface, board interactions, modals, and messenger panel.

Backend

Node.js with Express handles auth, projects, columns, tasks, comments, notifications, and static web serving.

Database

Prisma and SQLite store users, project memberships, tasks, comments, activity logs, and notifications.

Real time

Socket.IO keeps board refreshes, comments, member updates, and notifications live between collaborators.

Security

JWT protects API routes, bcrypt hashes passwords, and project membership checks gate project data.

04 / Requirements

What you need to run it.

  • Node.js installed on your machine.
  • Backend dependencies installed with npm.
  • Prisma database pushed to SQLite.
  • A browser opened at the Core server URL.
